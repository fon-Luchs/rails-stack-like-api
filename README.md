# Rails Stack Like API - Sample Application

## Install

### Clone repository
```
git clone https://github.com/Alexbaboshyn/rails-stack-like-api
```

### Install gems
```
gem install bundler
```

```
bundle install
```
### Create databases
```
rake db:create
```

### Run migrations
```
rake db:migrate
```

### Run specs
```
rake
```

### Run server
```
rails s
```
